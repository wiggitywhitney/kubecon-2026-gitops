# Scaling on Satisfaction — Slide Deck

Quarto + Reveal.js presentation for KubeCon EU 2026.

## Quick Start

### 1. Install Quarto

**macOS (Homebrew):**
```bash
brew install --cask quarto
```

If that requires sudo, use the tarball instead:
```bash
curl -L "https://github.com/quarto-dev/quarto-cli/releases/latest/download/quarto-macos.tar.gz" -o /tmp/quarto.tar.gz
tar -xzf /tmp/quarto.tar.gz -C ~/opt
export PATH="$HOME/opt/bin:$PATH"
```

### 2. Install Chromium (for Mermaid SVG rendering)

```bash
quarto install chromium
```

### 3. Preview

```bash
cd slides
quarto preview index.qmd
```

Opens a live preview in your browser that auto-refreshes as you edit.

**Keyboard shortcuts in the presentation:**
- Arrow keys: navigate slides
- **S**: open speaker notes view
- **F**: fullscreen
- **O**: slide overview grid

## Structure

- `index.qmd` — all slides (Quarto markdown)
- `_quarto.yml` — project config (Reveal.js output, Mermaid settings)
- `custom.scss` — teal/cyan accent theme
- `images/` — decorative PNGs
- `_output/` — rendered HTML (gitignored, generated by `quarto render`)

## Editing

Slides are plain markdown with Mermaid diagram blocks. Edit `index.qmd` and the preview auto-refreshes.

Speaker notes go inside `::: {.notes}` / `:::` blocks — only visible in speaker view (press S).
